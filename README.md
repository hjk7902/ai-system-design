# ai-system-design
Spring Boot AI 시스템 설계 - 컨텍스트 중심 AI 애플리케이션 구축

# 개발환경
## 자바
  - JDK 25
  - Sprint Tools for Eclipse 5.0.1
  - Spring Boot 3.5.10
  - Spring AI 1.1.2
## 파이썬
  - Python 3.13
  - VSCode
## MQTT 서버
  - Mosquitto

# 프로젝트 임포트
## 자바
  - JavaSpringProject.zip
  - 이클립스 -> File -> Import
  - General -> Existing Projects into Workspace -> Next
  - Select archive file: JavaSpringProject.zip 파일 선택 -> Finish
## 파이썬
  - vs-workspace.zip
  - 압축 풀고 VSCode에서 폴더를 열면 됨
